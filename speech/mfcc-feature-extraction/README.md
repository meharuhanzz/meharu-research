# MFCC Feature Extraction

This project demonstrates how to extract **Mel Frequency Cepstral Coefficients (MFCC)** from speech signals using Python.

## Overview
MFCC is one of the most widely used features in speech processing tasks such as:

- Automatic Speech Recognition (ASR)
- Speaker Identification
- Speech Emotion Recognition
- Audio Classification

This project performs the following steps:

1. Load an audio file
2. Visualize the waveform
3. Compute the spectrogram
4. Generate the Mel spectrogram
5. Extract MFCC features
6. Visualize MFCC coefficients

## Tools and Libraries

- Python
- Librosa
- NumPy
- Matplotlib
